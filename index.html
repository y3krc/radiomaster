What's inside

Sounds Universal

41 Y3K RC voice prompts — z-named, EdgeTX-spec PCM 16-bit mono 32 kHz. A sound is just a .wav, so it plays the same on any RadioMaster + EdgeTX radio.

zy3k_rc zcrawlers_roll_out zhit_the_trail zyou_got_this

Models Per-controller

Per-radio .yml model configs — switches, mixes, and channels encoded for a specific controller. Currently models/mt12/ (RadioMaster MT12); more controllers as they're added.

Use the sounds — any RadioMaster + EdgeTX

  1. Start from an EdgeTX SD card — set one up with EdgeTX Buddy.
  2. Copy sounds/*.wav into the card's SOUNDS/en/.
  3. Assign via Special / Global Functions — they all start with z, so they cluster at the bottom of the picker.

Use a model — your specific controller

  1. Grab the .yml for your radio from models/<controller>/.
  2. Copy it into the card's BACKUP/.
  3. On the radio, restore it from the model-select menu — safer than dropping straight into MODELS/.

Requirements

  • FirmwareEdgeTX
  • RadioAny RadioMaster radio
  • AudioSounds are PCM 16-bit mono 32 kHz (EdgeTX silently skips anything off-spec)
